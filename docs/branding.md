# Branding

## Product name

Primary name: **OrbitSync**

Positioning: "Local-first sync infrastructure for Flutter teams."

## Visual direction

- dark developer aesthetic
- orbital rings around a database glyph
- neon violet and cyan accents
- terminal-inspired diagrams and queue timelines

## Logo concepts

1. **Orbit database:** stacked database cylinder with two orbit rings.
2. **Delta comet:** triangular delta mark moving through an orbital path.
3. **Pulse queue:** waveform that becomes a sync arrow.

## Landing page sections

- Hero: "Offline sync as easy as Firestore."
- Feature grid: storage adapters, realtime, retries, conflicts, CRDTs.
- Architecture diagram.
- Mock server playground.
- Enterprise cloud CTA.
