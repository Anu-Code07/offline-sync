# Roadmap

- GraphQL sync transport with schema-aware delta patches.
- Peer-to-peer sync over local networks and WebRTC data channels.
- Edge sync nodes for low-latency regional collaboration.
- AI-assisted sync tuning for batch sizes and retry windows.
- Distributed cache invalidation and materialized local views.
- Rich collaborative editing primitives for notes, canvases, and documents.
- Hosted dashboard for queue depth, conflict rate, latency, and active devices.
